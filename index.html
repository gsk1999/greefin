🤖 Scan Preferences
Batch Scan Model
Tiered (recommended) — uses Haiku first (~$0.003/receipt), escalates to Sonnet only when fields are missing or receipt is complex. ~25× cheaper than Sonnet-only.
Sonnet only — maximum accuracy for every receipt. Best for difficult thermal/faded receipts.
Enhanced Item Lookup
After scanning, uses web search to identify full product names from receipt shortforms (e.g. HNTS PNTBTR 340G → Hunts Peanut Butter 340g). Best for grocery receipts. Adds ~5–10s and uses slightly more API credits.
Minimum Valid Year
If AI detects a date earlier than this year, it will be flagged as invalid and you'll be asked to set it manually. Catches OCR errors like reading 2023 instead of 2026.
— receipts dated before this will be flagged
Verify Item Names Online
After scanning, searches the web to verify and correct product names — especially useful for thermal receipts with smudged or misread text (e.g. 99 Speedmart, Giant). Runs alongside Enhanced Lookup if both are on. Adds ~5–8s per scan.
